# Expense Tracker

A simple Expense Tracker web application built using HTML, CSS, and JavaScript.  
The application allows users to add income and expense transactions, calculate total balance.

---

## Features

- Add income and expense transactions
- Automatically calculate balance
- Separate income and expense totals
- Delete transactions
- Store data using localStorage
- Dynamic DOM updates without page reload

---

## Technologies Used

- HTML
- CSS
- JavaScript

---

## How It Works

- Positive amounts are treated as income
- Negative amounts are treated as expenses
- Transactions are stored in localStorage so data remains after refreshing the page
